
> # Example config

Config file - **`some_file.json`**
</br>
*file JSON


***


``` js
{
    "Host":"localhost",
    "Port":"8080",
    "PathJournal":"logs/service.log"  
}
```

**Host** - your hostname (default: `localhost)`
</br>
**Port** - your port (default: `8080`)
</br>
**PathJournal** - path to logs (default: `./logs/service.log`)
